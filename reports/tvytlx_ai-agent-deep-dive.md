## 项目简介

### 项目定位

**tvytlx/ai-agent-deep-dive** 是一个专注于 AI Agent 技术源码分析的**教学研究项目**。该项目通过 Jupyter Notebook 的形式，系统性地深入分析主流 AI Agent 框架的核心源码，揭示其设计思想与实现机制。

项目的核心定位是：**源码分析 + 教学演示 + 实践指南**

### 学习路线设计

项目采用**渐进式教学结构**，设计了一条从浅入深的六大模块学习路径：

```
AI Agent 基础 → 记忆系统 → 工具调用 → 任务规划 → 反思机制 → Agentic RAG
```

### 核心模块概览

| 序号 | 模块名称 | 文件名 | 学习内容 |
|------|----------|--------|----------|
| 01 | AI Agent 基础 | `01_AI_Agent_intro.ipynb` | AI Agent 基本概念、发展历史、技术架构 |
| 02 | Agent 记忆 | `02_Agent_memory.ipynb` | 记忆系统设计（短期/长期记忆、检索机制） |
| 03 | Agent 工具 | `03_Agent_tools.ipynb` | 工具定义注册、选择策略、执行处理 |
| 04 | Agent 规划 | `04_Agent_planning.ipynb` | 任务分解、执行计划生成、计划调整 |
| 05 | Agent 反思 | `05_Agent_Reflection.ipynb` | 自我评估、错误纠正、性能优化 |
| 06 | Agentic RAG | `06_Agentic_RAG.ipynb` | 检索增强生成、动态知识更新、多跳推理 |

### 目标用户群体

本项目适合以下人群使用：

- ✅ AI 应用开发者，希望了解 Agent 技术的实现原理
- ✅ 对 Agent 技术感兴趣的学习者，从零开始系统学习
- ✅ 希望深入理解 Agent 框架源码的工程师
- ✅ 技术博主，寻找高质量的技术内容素材
- ✅ 培训讲师，构建 AI Agent 教学课程

#### 2. 深入框架内部实现

项目不仅教授如何使用 LangChain/LangGraph，更深入分析其源码实现机制：

- **LangChain 核心抽象**：Prompt 模板、Output Parser、Chain 机制
- **LangGraph 图执行**：状态机设计、节点路由、边条件判断
- **Memory 接口设计**：ConversationBufferMemory、VectorStoreMemory
- **Tool 调用机制**：Function Calling、Tool Selection Strategy
