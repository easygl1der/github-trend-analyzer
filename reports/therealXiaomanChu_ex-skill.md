## 二、项目简介

### 2.1 项目背景

**ex-skill** 是一个将聊天记录（如 Tinder、微信等）蒸馏成 AI Skill 的命令行工具。该项目受到 [colleague-skill](https://github.com/99s-work/colleague-skill) 的启发，旨在通过分析前任的聊天方式，将其对话风格转化为可执行的 AI Skill，使用户能够以"ta的方式"继续与 AI 进行模拟对话。

### 2.2 项目类型

**AI 应用工具（命令行工具 + 可扩展框架）**

项目具有以下核心特征：

- **命令行工具**：提供简洁的 CLI 界面供用户操作
- **可扩展框架**：支持自定义 Skill 和 Provider 的灵活扩展
- **数据驱动**：通过 CSV/JSON 格式导入聊天数据
- **多后端支持**：支持多种 LLM API 提供商（Coze、OpenAI、Claude 等）

### 2.3 核心功能

| 功能模块 | 说明 |
|----------|------|
| 数据导入 | 支持从 Tinder 导出 CSV 和微信 JSON 格式的聊天记录 |
| Skill 蒸馏 | 将聊天记录转换为可执行的 AI Skill |
| 多平台支持 | 通过 Docker 快速部署，支持多种 LLM API |
| 命令行界面 | 提供简洁易用的 CLI 工具 |
