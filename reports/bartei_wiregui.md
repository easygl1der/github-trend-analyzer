## 项目简介

**WireGui** 是一个受 **Firezone** 启发而开发的自托管 VPN 管理平台，旨在提供 Web 界面来管理 WireGuard VPN 服务器和 nftables 防火墙规则。

### 核心定位

该项目填补了自托管 VPN 管理工具的市场需求，为系统管理员提供了一个轻量级、功能明确的替代方案。用户无需依赖第三方云服务，即可通过友好的 Web 界面管理自己的 VPN 基础设施。

### 主要功能

1. **WireGuard VPN 管理**
   - 配置和管理 WireGuard 网络接口
   - 添加、修改、删除对等体（Peers）
   - 管理路由规则和隧道配置

2. **nftables 防火墙集成**
   - 与 WireGuard 深度集成的防火墙规则管理
   - NAT 配置
   - 访问控制策略

3. **嵌入式 Web UI**
   - 响应式设计，适配多种设备
   - 单文件 HTML/CSS/JavaScript 实现
   - 无需额外的前端构建流程

4. **RESTful API**
   - 提供完整的后端 API 接口
   - 支持程序化管理和集成

### 目标用户

- 需要自托管 WireGuard VPN 管理界面的系统管理员
- 不想使用第三方云服务的团队
- 需要内网 VPN 解决方案的企业
- 学习 Go 网络编程的开发者

## 潜在问题

### 1. 安全风险

| 风险项 | 严重程度 | 说明 | 建议 |
|--------|---------|------|------|
| **无内置认证** | 中高 | 默认无身份验证机制 | 生产环境需配置反向代理认证 |
| **命令注入风险** | 中 | 使用 `exec.Command` 调用系统命令 | 需严格验证所有输入参数 |
| **权限过大** | 低 | 需要 root 权限运行 | 这是设计决定，难以避免 |
| **敏感信息暴露** | 低 | 日志和错误消息可能包含敏感信息 | 审计日志输出，配置错误处理 |
| **无 TLS 默认支持** | 中 | HTTPS 需要额外配置 | 文档应强调生产环境使用 TLS |
